# МЕХАНИКА

### 1. Дальность полета тела (с высоты $h$, под углом $\alpha$)
**Используемые законы:** Кинематические уравнения равноускоренного движения (независимость движений по осям X и Y).

**Вывод:**
1.  Запишем уравнения движения в проекциях на оси (начало координат на земле под точкой броска):
    $X(t) = v_0 \cos\alpha \cdot t$
    $Y(t) = h + v_0 \sin\alpha \cdot t - \frac{gt^2}{2}$
2.  В момент падения $Y(t) = 0$. Решаем квадратное уравнение относительно времени полета $t_{pol}$:
    $\frac{g}{2}t_{pol}^2 - v_0 \sin\alpha \cdot t_{pol} - h = 0$
    $D = (v_0 \sin\alpha)^2 + 2gh$
    $t_{pol} = \frac{v_0 \sin\alpha + \sqrt{v_0^2 \sin^2\alpha + 2gh}}{g}$ (отрицательный корень отбрасываем).
3.  Подставляем $t_{pol}$ в уравнение для $X$:
    $L = v_0 \cos\alpha \cdot \frac{v_0 \sin\alpha + \sqrt{v_0^2 \sin^2\alpha + 2gh}}{g}$

### 2. Максимальная высота подъема (с высоты $h$, под углом $\alpha$)
**Используемые законы:** Кинематика, условие максимума высоты ($v_y = 0$).

**Вывод:**
1.  Скорость по вертикали: $v_y(t) = v_0 \sin\alpha - gt$.
2.  В верхней точке $v_y = 0 \Rightarrow t_{up} = \frac{v_0 \sin\alpha}{g}$.
3.  Подставляем это время в уравнение координаты $Y(t)$:
    $H_{max} = h + v_0 \sin\alpha \left(\frac{v_0 \sin\alpha}{g}\right) - \frac{g}{2}\left(\frac{v_0 \sin\alpha}{g}\right)^2$
    $H_{max} = h + \frac{v_0^2 \sin^2\alpha}{g} - \frac{v_0^2 \sin^2\alpha}{2g}$
    $H_{max} = h + \frac{v_0^2 \sin^2\alpha}{2g}$

### 3. Момент инерции полого толстостенного цилиндра
**Используемые законы:** Определение момента инерции $I = \int r^2 dm$, понятие плотности.

**Вывод:**
1.  Разобьем цилиндр на тонкие слои радиусом $r$ и толщиной $dr$.
2.  Объем слоя: $dV = 2\pi r L dr$. Масса слоя: $dm = \rho dV = 2\pi \rho L r dr$.
3.  Интегрируем от $R_2$ (внутр.) до $R_1$ (внешн.):
    $I = \int r^2 dm = \int_{R_2}^{R_1} r^2 (2\pi \rho L r) dr = 2\pi \rho L \int_{R_2}^{R_1} r^3 dr$
    $I = 2\pi \rho L \left[ \frac{r^4}{4} \right]_{R_2}^{R_1} = \frac{\pi \rho L}{2} (R_1^4 - R_2^4)$.
4.  Выразим через массу. Масса цилиндра $m = \rho V = \rho \pi L (R_1^2 - R_2^2)$.
    Заметим, что $R_1^4 - R_2^4 = (R_1^2 - R_2^2)(R_1^2 + R_2^2)$.
    $I = \frac{1}{2} [\pi \rho L (R_1^2 - R_2^2)] (R_1^2 + R_2^2) = \frac{1}{2} m (R_1^2 + R_2^2)$.

### 4. Момент инерции полого шара (толстостенной сферы)
**Используемые законы:** Аддитивность моментов инерции.
**Примечание:** Рассматриваем как разность сплошного шара радиуса $R_1$ и шара $R_2$.

**Вывод:**
1.  Для сплошного шара $I = \frac{2}{5} m R^2$. Но нам нужна плотность. $m = \rho \frac{4}{3}\pi R^3$.
    $I_{solid}(R) = \frac{2}{5} (\rho \frac{4}{3}\pi R^3) R^2 = \frac{8}{15}\pi \rho R^5$.
2.  Момент инерции полого = $I(R_1) - I(R_2)$:
    $I = \frac{8}{15}\pi \rho (R_1^5 - R_2^5)$.
3.  Выражаем через массу тела $m = \rho (V_1 - V_2) = \rho \frac{4}{3}\pi (R_1^3 - R_2^3) \Rightarrow \rho = \frac{m}{\frac{4}{3}\pi(R_1^3 - R_2^3)}$.
4.  Подставим $\rho$:
    $I = \frac{8}{15}\pi \frac{m}{\frac{4}{3}\pi(R_1^3 - R_2^3)} (R_1^5 - R_2^5) = \frac{2}{5} m \frac{R_1^5 - R_2^5}{R_1^3 - R_2^3}$.

### 5. Момент инерции диска относительно диаметра
**Используемые законы:** Теорема о перпендикулярных осях.

**Вывод:**
1.  Известен момент инерции диска относительно оси симметрии (перпендикулярно плоскости): $I_z = \frac{1}{2}mR^2$.
2.  Пусть оси $X$ и $Y$ лежат в плоскости диска (вдоль диаметров). По теореме: $I_z = I_x + I_y$.
3.  В силу симметрии диска $I_x = I_y$.
4.  $I_z = 2 I_{diam} \Rightarrow I_{diam} = \frac{I_z}{2}$.
    $I_{diam} = \frac{1}{2} \left( \frac{1}{2}mR^2 \right) = \frac{1}{4}mR^2$.

### 6. Момент инерции стержня относительно конца
**Используемые законы:** Прямое интегрирование (или теорема Штейнера).

**Вывод (через интеграл):**
1.  Линейная плотность $\lambda = m/l$. Элемент длины $dx$ на расстоянии $x$ от оси.
2.  $dm = \lambda dx$.
3.  $I = \int_0^l x^2 dm = \int_0^l x^2 \lambda dx = \frac{m}{l} \int_0^l x^2 dx$.
    $I = \frac{m}{l} \left[ \frac{x^3}{3} \right]_0^l = \frac{m}{l} \frac{l^3}{3} = \frac{1}{3}ml^2$.

# МЕХАНИЧЕСКИЕ КОЛЕБАНИЯ

### 7. Период маятника в движущейся системе (с ускорением $a$)
**Используемые законы:** Второй закон Ньютона, принцип эквивалентности (эффективное ускорение).

**Вывод:**
1.  На груз в неинерциальной системе отсчета (НСО) действует сила тяжести $m\vec{g}$ и сила инерции $\vec{F}_{in} = -m\vec{a}$.
2.  Эффективная сила тяжести: $m\vec{g}_{eff} = m(\vec{g} - \vec{a})$.
3.  Эффективное ускорение свободного падения: $g' = |\vec{g} - \vec{a}|$.
4.  Формула периода математического маятника: $T = 2\pi \sqrt{\frac{l}{g_{eff}}}$.
    (Векторная разность берется в зависимости от направления $\vec{a}$).

### 8. Период физического маятника (стержень, точка подвеса на расст. $x$ от конца)
**Используемые законы:** Основное уравнение вращательного движения, теорема Штейнера.

**Вывод:**
1.  Центр масс стержня находится в середине ($L/2$). Расстояние от точки подвеса до центра масс: $d = |\frac{L}{2} - x|$.
2.  Момент инерции относительно центра масс $I_c = \frac{1}{12}mL^2$.
3.  Момент инерции относительно точки подвеса (т. Штейнера): $I = I_c + md^2 = \frac{1}{12}mL^2 + m(\frac{L}{2}-x)^2$.
4.  Период физического маятника: $T = 2\pi \sqrt{\frac{I}{mgd}}$.
    $T = 2\pi \sqrt{\frac{\frac{1}{12}mL^2 + m(\frac{L}{2}-x)^2}{mg|\frac{L}{2}-x|}}$.

### 9. Период пружинного маятника (последовательные пружины $k_1, k_2$)
**Используемые законы:** Закон Гука, сложение деформаций.

**Вывод:**
1.  При последовательном соединении сила упругости одна и та же: $F = k_{eq} \Delta x_{total}$.
2.  Общее удлинение $\Delta x_{total} = \Delta x_1 + \Delta x_2$.
3.  $F/k_{eq} = F/k_1 + F/k_2 \Rightarrow \frac{1}{k_{eq}} = \frac{1}{k_1} + \frac{1}{k_2} \Rightarrow k_{eq} = \frac{k_1 k_2}{k_1 + k_2}$.
4.  Период колебаний: $T = 2\pi \sqrt{\frac{m}{k_{eq}}} = 2\pi \sqrt{\frac{m(k_1 + k_2)}{k_1 k_2}}$.

### 10. Уравнение циклической частоты затухающих колебаний
**Используемые законы:** Второй закон Ньютона с силой вязкого трения ($F_{tr} = -rv$).

**Вывод:**
1.  Уравнение движения: $ma = -kx - rv \Rightarrow m\ddot{x} + r\dot{x} + kx = 0$.
2.  Делим на $m$, вводим $\omega_0^2 = k/m$ и $2\beta = r/m$:
    $\ddot{x} + 2\beta\dot{x} + \omega_0^2 x = 0$.
3.  Ищем решение в виде $x = A_0 e^{-\beta t} \cos(\omega t + \phi)$.
4.  Характеристическое уравнение: $\lambda^2 + 2\beta\lambda + \omega_0^2 = 0$.
5.  Корни: $\lambda = -\beta \pm \sqrt{\beta^2 - \omega_0^2}$.
6.  Для колебательного режима ($\omega_0 > \beta$) подкоренное выражение отрицательно, $\sqrt{\dots} = i\sqrt{\omega_0^2 - \beta^2}$.
7.  Циклическая частота затухающих колебаний: $\omega = \sqrt{\omega_0^2 - \beta^2}$.

### 11. Уравнение резонансной частоты (через декремент $\delta$)
**Используемые законы:** Анализ амплитуды вынужденных колебаний на экстремум.

**Вывод:**
1.  Амплитуда при вынужденных колебаниях: $A(\omega) = \frac{F_0/m}{\sqrt{(\omega_0^2 - \omega^2)^2 + 4\beta^2 \omega^2}}$.
2.  Для резонанса нужно минимизировать подкоренное выражение. Берем производную по $\omega$ от знаменателя и приравниваем к нулю:
    $\frac{d}{d\omega} [(\omega_0^2 - \omega^2)^2 + 4\beta^2 \omega^2] = 2(\omega_0^2 - \omega^2)(-2\omega) + 8\beta^2 \omega = 0$.
    $-4\omega(\omega_0^2 - \omega^2) + 8\beta^2 \omega = 0$.
    $-\omega_0^2 + \omega^2 + 2\beta^2 = 0 \Rightarrow \omega_{res} = \sqrt{\omega_0^2 - 2\beta^2}$.
3.  Связь $\beta$ и логарифмического декремента затухания $\delta$: $\delta = \beta T \approx \beta \frac{2\pi}{\omega_0}$ (при малом затухании). Отсюда $\beta \approx \frac{\delta \omega_0}{2\pi}$.
    Точная формула $\delta = \beta T = \beta \frac{2\pi}{\sqrt{\omega_0^2-\beta^2}}$.
    Если использовать точную, вывод сложен, обычно в курсах просят выразить $\beta$ через $\delta$.
    Ответ: $\omega_{res} = \sqrt{\omega_0^2 - 2\beta^2}$, где $\beta$ выражается через $\delta$.

### 12. Уравнение Циолковского с учетом силы тяжести
**Используемые законы:** Второй закон Ньютона в импульсной форме (уравнение Мещерского).

**Вывод:**
1.  Внешние силы: $\vec{F}_{ext} = m\vec{g}$ (направлена вниз).
2.  Уравнение движения ракеты: $m \frac{d\vec{v}}{dt} = \vec{u} \frac{dm}{dt} + m\vec{g}$, где $\vec{u}$ — скорость истечения газов относительно ракеты (вниз), $\vec{v}$ — скорость ракеты (вверх).
3.  В проекции на вертикальную ось (вверх):
    $m \frac{dv}{dt} = -u \frac{dm}{dt} - mg$ (здесь $u$ — модуль скорости истечения, $dm/dt$ отрицательно, но в формуле Мещерского член реактивной силы $-u \frac{dm}{dt}$ положителен, так как масса убывает. Строже: $F_{reac} = u \cdot |\frac{dm}{dt}|$. Пусть $\mu = -dm/dt$. Тогда $m dv/dt = \mu u - mg$).
    Разделяем переменные: $dv = -u \frac{dm}{m} - g dt$.
4.  Интегрируем от $t=0$ до $t$:
    $v - v_0 = -u \ln m \Big|_{m_0}^{m} - gt = u \ln \frac{m_0}{m} - gt$.
    $v(t) = v_0 + u \ln \left(\frac{m_0}{m(t)}\right) - gt$.

# МОЛЕКУЛЯРНАЯ ФИЗИКА

### 13. Вывод основного уравнения МКТ
**Задание:** Вывод уравнения МКТ с помощью уравнения состояния (или связь температуры и энергии).
**Используемые законы:** Основное уравнение МКТ (давление), Уравнение Менделеева-Клапейрона.

**Вывод:**
1.  Основное уравнение МКТ (из механики соударений): $P = \frac{2}{3} n \bar{E}_k$, где $n = N/V$ — концентрация.
    $P = \frac{2}{3} \frac{N}{V} \bar{E}_k \Rightarrow PV = \frac{2}{3} N \bar{E}_k$.
2.  Уравнение состояния идеального газа: $PV = \nu RT = \frac{N}{N_A} RT = N k T$ (так как $R/N_A = k$).
3.  Приравниваем правые части:
    $\frac{2}{3} N \bar{E}_k = N k T$.
4.  Сокращаем $N$:
    $\bar{E}_k = \frac{3}{2} kT$ (Связь средней кинетической энергии и температуры).
    Подставив обратно в первое уравнение, получаем $P = nkT$.

### 14. Наиболее вероятная скорость (из распределения Максвелла)
**Используемые законы:** Распределение Максвелла, поиск экстремума функции.

**Вывод:**
1.  Функция распределения: $F(v) = A \cdot v^2 e^{-\frac{m_0 v^2}{2kT}}$.
2.  Чтобы найти максимум, берем производную по $v$ и приравниваем к 0:
    $\frac{dF}{dv} = A \left( 2v \cdot e^{-\dots} + v^2 \cdot e^{-\dots} \cdot \left( -\frac{2m_0 v}{2kT} \right) \right) = 0$.
3.  Сокращаем на $A \cdot v \cdot e^{-\dots}$ (так как $v \neq 0, \infty$):
    $2 - v \cdot \frac{m_0 v}{kT} = 0$.
    $2 = \frac{m_0 v^2}{kT} \Rightarrow v_{ver}^2 = \frac{2kT}{m_0}$.
4.  $v_{ver} = \sqrt{\frac{2kT}{m_0}} = \sqrt{\frac{2RT}{M}}$.

### 15. Средняя квадратичная скорость
**Используемые законы:** Определение среднего значения функции.

**Вывод:**
1.  Определение: $\bar{v^2} = \int_0^\infty v^2 F(v) dv$.
2.  Используя $F(v) = 4\pi \left(\frac{m_0}{2\pi kT}\right)^{3/2} v^2 e^{-\alpha v^2}$, где $\alpha = \frac{m_0}{2kT}$.
    $\bar{v^2} = C \int_0^\infty v^4 e^{-\alpha v^2} dv$.
3.  Табличный интеграл $\int_0^\infty x^4 e^{-ax^2} dx = \frac{3}{8a^2}\sqrt{\frac{\pi}{a}}$.
    После подстановки и сокращений получается: $\bar{v^2} = \frac{3kT}{m_0}$.
4.  $v_{kv} = \sqrt{\bar{v^2}} = \sqrt{\frac{3kT}{m_0}} = \sqrt{\frac{3RT}{M}}$.

### 16. Изменение энтропии (Закон Бойля-Мариотта / Изотермический)
**Используемые законы:** Первое начало термодинамики, определение энтропии.

**Вывод:**
1.  $T = const \Rightarrow dU = 0$.
2.  $dQ = dA = P dV$.
3.  $dS = \frac{dQ}{T} = \frac{P dV}{T}$.
4.  Из $PV = \nu RT \Rightarrow \frac{P}{T} = \frac{\nu R}{V}$.
5.  $dS = \nu R \frac{dV}{V}$. Интегрируем:
    $\Delta S = \nu R \ln \frac{V_2}{V_1}$ (или через давление $\nu R \ln \frac{P_1}{P_2}$).

### 17. Изменение энтропии (Закон Шарля / Изохорный)
**Примечание:** В русскоязычной литературе закон Шарля — это изохорный процесс ($V=const$).
**Вывод:**
1.  $V = const \Rightarrow dA = 0$.
2.  $dQ = dU = \frac{m}{M} C_V dT$.
3.  $dS = \frac{dQ}{T} = \frac{m}{M} C_V \frac{dT}{T}$.
4.  Интегрируем:
    $\Delta S = \frac{m}{M} C_V \ln \frac{T_2}{T_1}$.

### 18. Изменение энтропии (Закон Гей-Люссака / Изобарный)
**Примечание:** В русскоязычной литературе закон Гей-Люссака — это изобарный процесс ($P=const$).
**Вывод:**
1.  $P = const$. Теплоемкость при постоянном давлении $C_P$.
2.  $dQ = \frac{m}{M} C_P dT$.
3.  $dS = \frac{dQ}{T} = \frac{m}{M} C_P \frac{dT}{T}$.
4.  Интегрируем:
    $\Delta S = \frac{m}{M} C_P \ln \frac{T_2}{T_1}$.

### 19. Работа адиабатного процесса (через $\gamma$)
**Используемые законы:** Уравнение Пуассона $PV^\gamma = const$.

**Вывод:**
1.  Работа $A = \int_{V_1}^{V_2} P dV$.
2.  $P = \frac{C}{V^\gamma}$, где $C = P_1 V_1^\gamma$.
3.  $A = C \int_{V_1}^{V_2} V^{-\gamma} dV = C \left[ \frac{V^{1-\gamma}}{1-\gamma} \right]_{V_1}^{V_2}$.
4.  $A = \frac{1}{1-\gamma} (P_2 V_2^\gamma \cdot V_2^{1-\gamma} - P_1 V_1^\gamma \cdot V_1^{1-\gamma})$. (Использовали $C$ как параметры соответствующих точек).
5.  $A = \frac{P_2 V_2 - P_1 V_1}{1-\gamma} = \frac{P_1 V_1 - P_2 V_2}{\gamma - 1}$.
6.  Через температуры: $A = \frac{\nu R (T_1 - T_2)}{\gamma - 1}$.

### 20. Максимальный КПД тепловой машины (Цикл Карно)
**Используемые законы:** Определение КПД, уравнения адиабаты.

**Вывод:**
1.  $\eta = \frac{Q_1 - Q_2}{Q_1} = 1 - \frac{Q_2}{Q_1}$.
2.  $Q_1 = \nu R T_1 \ln(V_2/V_1)$ (изотерма $T_1$).
3.  $Q_2 = \nu R T_2 \ln(V_3/V_4)$ (изотерма $T_2$, отданное тепло).
4.  Отношение: $\frac{Q_2}{Q_1} = \frac{T_2 \ln(V_3/V_4)}{T_1 \ln(V_2/V_1)}$.
5.  Для адиабат ($2\to3$ и $4\to1$): $T_1 V_2^{\gamma-1} = T_2 V_3^{\gamma-1}$ и $T_1 V_1^{\gamma-1} = T_2 V_4^{\gamma-1}$.
6.  Делим одно на другое: $(V_2/V_1)^{\gamma-1} = (V_3/V_4)^{\gamma-1} \Rightarrow V_2/V_1 = V_3/V_4$.
7.  Логарифмы сокращаются:
    $\eta_{max} = 1 - \frac{T_2}{T_1} = \frac{T_1 - T_2}{T_1}$.

### 21. Максимальный КПД (Холодильный коэффициент) холодильной машины
**Используемые законы:** Определение COP (coefficient of performance), обратный цикл Карно.

**Вывод:**
1.  Холодильный коэффициент $\varepsilon = \frac{Q_{cold}}{A}$ (тепло, забранное у холодильника, делить на затраченную работу).
2.  $A = Q_{hot} - Q_{cold}$ (по закону сохранения энергии).
3.  $\varepsilon = \frac{Q_2}{Q_1 - Q_2}$.
4.  Для цикла Карно справедливо соотношение $\frac{Q_1}{T_1} = \frac{Q_2}{T_2}$ (приведенная теплота равна).
    $Q_1 = Q_2 \frac{T_1}{T_2}$.
5.  $\varepsilon = \frac{Q_2}{Q_2 \frac{T_1}{T_2} - Q_2} = \frac{1}{\frac{T_1}{T_2} - 1}$.
    $\varepsilon = \frac{T_2}{T_1 - T_2}$, где $T_2$ — температура камеры (холода), $T_1$ — температура среды (нагревателя).
